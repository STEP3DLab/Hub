# Проверка качества v2

1. Структура соответствует стандартным свадебным сайтам: приглашение, календарь, countdown, программа, локации, дресс-код, подарки, анкета, FAQ, финал.
2. Все крупные разделы отделены отдельными `<section>` и визуальными переходами.
3. На сайте нет фотографий людей и коллажей с моделями.
4. В левом верхнем углу нет инициалов; стоит только дата.
5. Видимые имена пары убраны из сайта.
6. Имена гостей выводятся персонально по параметру `?guest=`.
7. Гостевые имена оформлены отдельным рукописным шрифтом `Marck Script` с fallback-цепочкой.
8. Основные заголовки — `Cormorant Garamond`, текст — `Manrope`.
9. Дресс-код представлен только круглыми цветовыми свотчами.
10. Есть разные сценарии программы: `full_day`, `banquet_only`, `child`.
11. Алкогольная анкета скрывается для `child` и для неизвестной ссылки.
12. Карты открываются через Яндекс.Карты.
13. Есть подготовленный Apps Script для записи анкеты в Google Таблицу.
14. Код не содержит явных синтаксических ошибок по базовой проверке структуры.
15. Мобильный UX учтен: крупные кнопки, mobile menu, адаптивная сетка, читаемые размеры.
