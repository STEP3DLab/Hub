# Бесконечность - интерактивный сайт

Автономный образовательный сайт о бесконечности.

## Что внутри

- один HTML-файл без внешних библиотек;
- адаптивный дизайн для телефона и компьютера;
- темная и светлая тема;
- прогресс чтения;
- подсветка активного раздела;
- интерактивный счетчик "прибавить 1";
- интерактивный отель Гильберта;
- анимированная инфографика;
- разделы: математика, геометрия, физика, космос, философия, религия, искусство, программирование.

## Как открыть

Откройте `index.html` в браузере.

## Для GitHub Pages

Если GitHub Pages включен для репозитория `STEP3DLab/Hub` из ветки `main`, страница будет доступна по адресу:

`https://step3dlab.github.io/Hub/infinity/`

Если страница не открывается, нужно включить GitHub Pages в настройках репозитория: Settings -> Pages -> Source: Deploy from a branch -> Branch: main -> Folder: /root.
